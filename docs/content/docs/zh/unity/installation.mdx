# 安装

有三种方法可以在您的项目中安装 Developerworks Unity SDK。

## 方法 1：Unity Package Manager（推荐）

这是最简单且推荐的安装 SDK 的方法。

1. 打开 Unity Package Manager：`Window > Package Manager`
2. 点击左上角的 `+` 按钮
3. 选择 `Add package from git URL...`
4. 输入以下 URL：

```
https://github.com/cnqdztp/Developerworks-UnitySDK.git
```

5. 点击 `Add` 并等待 Unity 下载并导入包

## 方法 2：下载并导入

如果您更喜欢手动安装：

1. 访问 [GitHub Releases 页面](https://github.com/cnqdztp/Developerworks-UnitySDK/releases)
2. 下载最新的发布包
3. 将内容解压到 Unity 项目的 `Assets/Developerworks_SDK/` 目录
4. Unity 将自动导入 SDK

## 方法 3：Git 子模块

适用于希望通过 Git 保持 SDK 更新的高级用户：

```bash
cd your-unity-project
git submodule add https://github.com/cnqdztp/Developerworks-UnitySDK.git Assets/Developerworks_SDK
```

## 验证安装

安装后，验证 SDK 是否正确安装：

1. 在 Unity 中，导航到 `Assets/Developerworks_SDK/`
2. 您应该看到以下文件夹：
   - `Prefab/`
   - `Scripts/`
   - `Examples/`

## 下一步

现在您已经安装了 SDK，请继续阅读[快速开始](/zh/docs/unity/quick-start)指南来设置您的第一个集成。
