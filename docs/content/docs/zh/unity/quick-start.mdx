# 快速开始

本指南将帮助您在几分钟内将 Developerworks SDK 集成到您的 Unity 项目中。

## 前置条件

开始之前，请确保您已：

- 安装了 SDK（请参阅[安装](/zh/docs/unity/installation)）
- 从 [Developerworks 平台](https://developerworks.agentlandlab.com)获取了 Game ID

## 步骤 1：将 SDK 添加到场景

1. 在 `Assets/Developerworks_SDK/Prefab/DW_SDK.prefab` 中找到 `DW_SDK` 预制体
2. 将其拖入您的**第一个场景**（游戏启动时加载的场景）
3. 在层级视图中选择 `DW_SDK` GameObject
4. 在检查器中，填写您的 `Game ID`
5. （可选）为聊天和图像生成设置默认模型名称

## 步骤 2：初始化 SDK

创建新脚本或在场景中使用现有脚本：

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;

public class GameManager : MonoBehaviour
{
    async void Start()
    {
        // 开发模式：使用开发者令牌（会花钱！）
        bool success = await DeveloperworksSDK.Instance.InitializeDevelopment("your-developer-token");

        // 或生产模式：让玩家登录（免费！）
        // bool success = await DeveloperworksSDK.Instance.InitializeProduction();

        if (success)
        {
            Debug.Log("SDK 初始化成功！");
        }
        else
        {
            Debug.LogError("SDK 初始化失败");
        }
    }
}
```

<Callout type="warn">
  **警告**：开发者令牌会花钱！在发布的游戏中使用生产模式。
</Callout>

## 步骤 3：使用 AI 功能

### 聊天示例

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;

public class ChatExample : MonoBehaviour
{
    async void TestChat()
    {
        var chatClient = DeveloperworksSDK.Instance.Chat;

        var result = await chatClient.SendMessageAsync(
            "你好，AI！",
            modelName: "gpt-4o-mini"
        );

        if (result.success)
        {
            Debug.Log($"AI 回复：{result.data.message}");
        }
    }
}
```

### NPC 对话示例

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;

public class NPCExample : MonoBehaviour
{
    private NPCClient npcClient;

    void Start()
    {
        // 创建具有个性的 NPC
        npcClient = DeveloperworksSDK.Instance.CreateNPC(
            npcId: "village_elder",
            systemPrompt: "你是一位智慧的村长，总是用谜语说话。",
            modelName: "gpt-4o-mini"
        );
    }

    async void TalkToNPC()
    {
        var result = await npcClient.SendMessageAsync("你有什么智慧要分享？");

        if (result.success)
        {
            Debug.Log($"NPC 说：{result.data.message}");
        }
    }
}
```

### 图像生成示例

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;
using UnityEngine.UI;

public class ImageGenExample : MonoBehaviour
{
    public RawImage displayImage;

    async void GenerateImage()
    {
        var imageClient = DeveloperworksSDK.Instance.Image;

        var result = await imageClient.GenerateImageAsync(
            prompt: "一个有发光蘑菇的魔法森林",
            modelName: "stable-diffusion-3.5-large"
        );

        if (result.success && result.data.texture != null)
        {
            displayImage.texture = result.data.texture;
        }
    }
}
```

## 常见问题

### SDK 无法初始化

确保 `DW_SDK` 预制体在您的第一个场景中，并且具有有效的 Game ID。

### 身份验证错误

- 在开发模式下：检查您的开发者令牌是否有效
- 在生产模式下：确保玩家已成功登录

### 平台特定问题

- **WebGL**：某些功能如麦克风录音需要用户权限
- **macOS**：麦克风访问需要在构建设置中设置适当的权限
