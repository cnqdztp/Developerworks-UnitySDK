# Developerworks Unity SDK

Developerworks AI 平台官方 Unity SDK。将强大的 AI 功能（包括聊天、图像生成、语音转录和 NPC 对话）集成到您的 Unity 游戏中。

## 特性

- **🤖 AI 聊天**：使用各种 LLM 模型（GPT-4、GPT-4o-mini 等）进行文本生成和对话
- **🎨 图像生成**：具有可自定义参数的 AI 图像创建功能
- **🎤 语音转录**：使用 Whisper 模型进行语音转文字转换
- **💬 NPC 对话**：简化的游戏角色对话界面，具有自动历史管理功能
- **👤 玩家管理**：用户身份验证、点数跟踪和玩家信息
- **🎙️ 语音活动检测（VAD）**：智能麦克风录音，具有自动静音检测功能
- **🌐 跨平台**：支持 Windows、macOS、WebGL 和独立构建
- **🔒 安全认证**：基于令牌的身份验证，具有加密本地存储
- **⚡ 异步/等待**：基于 UniTask 的异步操作，实现流畅的游戏体验

## 系统要求

- **Unity**：2020.3 或更高版本
- **平台**：Windows、macOS、WebGL 或独立构建
- **.NET**：.NET Standard 2.0+
- **依赖项**（已包含）：
  - UniTask (Cysharp.Threading.Tasks)
  - Newtonsoft.Json (JSON.NET)

## 版本

当前版本：**v0.1.7.2-beta**
