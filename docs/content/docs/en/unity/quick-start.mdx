# Quick Start

This guide will help you integrate the Developerworks SDK into your Unity project in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- Installed the SDK (see [Installation](/en/docs/unity/installation))
- A Game ID from the [Developerworks Platform](https://developerworks.agentlandlab.com)

## Step 1: Add SDK to Your Scene

1. Locate the `DW_SDK` prefab in `Assets/Developerworks_SDK/Prefab/DW_SDK.prefab`
2. Drag it into your **first scene** (the scene that loads when your game starts)
3. Select the `DW_SDK` GameObject in the hierarchy
4. In the Inspector, fill in your `Game ID`
5. (Optional) Set default model names for chat and image generation

## Step 2: Initialize the SDK

Create a new script or use an existing one in your scene:

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;

public class GameManager : MonoBehaviour
{
    async void Start()
    {
        // Development mode: Use developer token (COSTS MONEY!)
        bool success = await DeveloperworksSDK.Instance.InitializeDevelopment("your-developer-token");

        // OR Production mode: Let players log in (FREE!)
        // bool success = await DeveloperworksSDK.Instance.InitializeProduction();

        if (success)
        {
            Debug.Log("SDK initialized successfully!");
        }
        else
        {
            Debug.LogError("Failed to initialize SDK");
        }
    }
}
```

  **Warning**: Developer tokens cost money! Use production mode in your released games.

## Step 3: Use AI Features

### Chat Example

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;

public class ChatExample : MonoBehaviour
{
    async void TestChat()
    {
        var chatClient = DeveloperworksSDK.Instance.Chat;

        var result = await chatClient.SendMessageAsync(
            "Hello, AI!",
            modelName: "gpt-4o-mini"
        );

        if (result.success)
        {
            Debug.Log($"AI Response: {result.data.message}");
        }
    }
}
```

### NPC Conversation Example

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;

public class NPCExample : MonoBehaviour
{
    private NPCClient npcClient;

    void Start()
    {
        // Create an NPC with personality
        npcClient = DeveloperworksSDK.Instance.CreateNPC(
            npcId: "village_elder",
            systemPrompt: "You are a wise village elder who speaks in riddles.",
            modelName: "gpt-4o-mini"
        );
    }

    async void TalkToNPC()
    {
        var result = await npcClient.SendMessageAsync("What is your wisdom?");

        if (result.success)
        {
            Debug.Log($"NPC says: {result.data.message}");
        }
    }
}
```

### Image Generation Example

```csharp
using Developerworks_SDK;
using Cysharp.Threading.Tasks;
using UnityEngine;
using UnityEngine.UI;

public class ImageGenExample : MonoBehaviour
{
    public RawImage displayImage;

    async void GenerateImage()
    {
        var imageClient = DeveloperworksSDK.Instance.Image;

        var result = await imageClient.GenerateImageAsync(
            prompt: "A magical forest with glowing mushrooms",
            modelName: "stable-diffusion-3.5-large"
        );

        if (result.success && result.data.texture != null)
        {
            displayImage.texture = result.data.texture;
        }
    }
}
```

## Common Issues

### SDK Not Initializing

Make sure the `DW_SDK` prefab is in your first scene and has a valid Game ID.

### Authentication Errors

- In development mode: Check that your developer token is valid
- In production mode: Make sure the player has successfully logged in

### Platform-Specific Issues

- **WebGL**: Some features like microphone recording require user permission
- **macOS**: Microphone access requires the proper permissions in build settings
